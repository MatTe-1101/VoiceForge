# VoiceForge

VoiceForge is an open-source Android application that transforms your smartphone into a professional real-time wireless microphone.

## Features

- 🎤 Live microphone
- 🔊 Real-time amplification
- 🎚 Professional mixer
- 🎵 Voice effects
- 🎼 Real-time AutoTune
- 🎧 Bluetooth support
- 📡 Wi-Fi streaming
- 💾 Presets
- 🎨 Material You UI
- 🌙 Dark Mode

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM
- Clean Architecture
- Hilt
- Room
- DataStore
- Oboe
